Módulos deste sistema


Listagem de Projetos

Na Listagem dos Projetos você visualiza todos os projetos de leis ativos e finalizados de seu município, a idéia é que você possa interagir com o seu voto e opinião a respeito de todos os projetos, fazendo com que o legislativo tenha um feedback efetivo sobre os projetos propostos.

Perfis dos Políticos

A página de "perfis dos políticos" permite que você tenha acesso a uma lista de todos os políticos da região, sendo assim, você pode visualizar de maneira simples todos os projetos que ele fez durante o seu mandato.

Desenvolvimento do Projeto



Este é um projeto de código-aberto e o código-fonte encontra-se no GitHub . O objetivo dessa iniciativa é que programadores de outras cidades possam utilizar este projeto para melhorar a democracia de sua região, pois se atuarmos na esfera municipal, o país melhora como um todo.

Este projeto é COMPLETAMENTE independente e não é financiado por NENHUM órgão público, partidos e não possue qualquer tipo de vínculo político. Eu sou apenas um programador Nerd que resolveu contribuir de maneira efetiva para a sociedade.

Tive como inspiração sistemas como o Vote na Web e Ranking dos Políticos, porém, estes dois projetos tem como objetivo a esfera federal de nossa política, dado que os dois projetos são fechados e não atuam nas esferas municipais resolvi criar este sistema de código-aberto.

Por um país efetivamente democrático!
